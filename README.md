### 🚀 About Me
I'm a performance-driven full-stack & systems developer passionate about building high-throughput backends, mission-critical industrial desktop software, and immersive graphics ecosystems. I specialize in engineering zero-allocation data pipelines, robust hybrid web-native architectures, and highly observable microservices.

- ⚙️ **Industrial Desktop & Systems:** Deeply versed in **WPF (.NET 8)** and modern MVVM architectures. Expert in crafting zero-allocation protocol parsers (`ArrayPool`, `ReadOnlySpan`) and asynchronous backpressure-controlled serial streams via `BoundedChannel`.
- 🌐 **Hybrid & Full-Stack Infrastructure:** Proficient in **WebView2 Hybrid development**, designing high-performance bi-directional C#/Web RPC bridges and real-time streaming layers. Experienced in building unified type-safe ecosystems using **Vue 3, TypeScript, and Zod**.
- ⚡ **High-Frequency Data & Observability:** Experienced in high-concurrency **ASP.NET Core** microservices and real-time distributed data distribution pipelines using **ZeroMQ (Pub/Sub)**. Skilled in time-series database optimization (**TimescaleDB**) and robust system observability (**Serilog**).
- 🎮 **Graphics & XR Mechanics:** Actively developing graphics software and immersive VR mechanics using **WebGL**, **Unity (C#)**, and the **XR Interaction Toolkit**.
